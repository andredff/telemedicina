Mapa do produto

Da compra até a operação administrativa.

Esta é a visão macro: o pedido nasce no checkout, vira dado no Supabase, dispara logística e depois fica visível para acompanhamento administrativo e para o cliente.

1

Paciente compra

Seleciona medicamentos, endereço, frete e pagamento.

2

Pedido nasce

O checkout grava o pedido em orders.

3

OS logística

O backend cria ou garante a OS e tenta notificar a equipe.

4

Admin acompanha

Pedidos, status, rastreio, eventos e histórico ficam no painel.

5

Financeiro consolida

MRR e pedidos avulsos entram no painel financeiro.

Mensagem-chave: não procure "o sistema" em um único botão. Ele é a sequência checkout -> pedido -> OS -> admin -> rastreio -> financeiro.

Sistema 3

Rastreamento: manual no admin, automático quando Correios estiver configurado.

O admin digita o código, mas ele não é mais um texto livre sem controle: o sistema valida formato/dígito, consulta os Correios quando há credenciais e atualiza status/eventos.

A

Admin informa código

Exemplo: AA123456785BR.

B

Validação local

Formato e dígito verificador reduzem erro de digitação.

C

Consulta Correios

Com token/contrato, busca eventos oficiais.

D

Cliente acompanha

Status, link dos Correios e eventos aparecem no pedido.

Configuração

Admin > Configurações > Correios: token, usuário, senha, cartão de postagem, contrato, CEP origem, PAC e SEDEX.

Status automático

Eventos como postado/em trânsito/entregue podem mover o pedido para enviado ou entregue.

Fallback

Sem credenciais do cliente, o código válido fica salvo e o link oficial dos Correios continua disponível.

Sistema 4

Acompanhamento financeiro consolidado.

O painel financeiro não é só contagem de status. Ele combina receita recorrente de assinaturas com pedidos avulsos de medicamentos, mantendo indicadores operacionais para leitura rápida.

MRR + Pedidos avulsos = Receita consolidada
O que o painel responde

Quanto entra por recorrência? Quanto veio de pedidos de medicamentos? Qual ticket médio consolidado? Qual churn e inadimplência?
